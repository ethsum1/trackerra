What is Trackerra?

Trackerra is an online software, for students, by students, making it easier than ever tolog, access, and analyseyour school test results. This software is perfect for the small topic tests you do in lessons, as you can see which topics you did better on, and which topics you need to revise more. Trackerra is great for the run-up to GCSEs and A-Levels because you can see what you did well in, and what needs more work on.

Easy Input

Trackerra allows you to easily input your exam data, wether it be a topic test or a mock, we have you covered. There's easy access from being able to use the grade 1-9 system, and input the test's respective subject and a little description. There is a detailed analysis of how you have been performing, over all your tests, or broken down by subject.
